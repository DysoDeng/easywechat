# 账号接入

如果你想使用本项目接入多个公众号，在本程序中，您可以为每个帐号都设置一个id，此id对应了该帐号的appid、token等信息。 如下表

idappIdsecret其它...
1wx3cf0f39249eb0e60f28f735d4f1c242f4687abb469072a29...
2wx49eb0e63cf0f39s28f735d4687abb469f1c2422a29f4f207...
Nwx5cfeb0e60f39249035f8f27d46f1c242f487a9072a29bb46...

在微信公众平台的设置中，您可以将您帐号中平台的 url 设置为 您的网址/?id=xxx，如:

http://www.example.com/wechat?id=1
1

而在程序入口处，根据 id 查找对应帐号的 appid 和 其它信息， 传入 'Overtrue\Wechat\Server'，完成初始化。
