# EasyWeChat

EasyWeChat 是一个开源的 微信open in new window 非官方 SDK。

EasyWeChat 的安装非常简单，因为它是一个标准的 Composeropen in new window 包，这意味着任何满足下列安装条件的 PHP 项目支持 Composer 都可以使用它。

# 环境需求

  • PHP >= 8.0
  • PHP cURL 扩展open in new window
  • PHP OpenSSL 扩展open in new window
  • PHP SimpleXML 扩展open in new window
  • PHP fileinfo 拓展open in new window

# 开始之前

本 SDK 不是一个全新再造的东西，你完全有必要在使用本 SDK 前做好以下工作：

  • 熟悉 PHP 常见的知识：自动加载、Composer 的使用、JSON 处理、cURL 的使用等；
  • 仔细阅读并看懂 微信官方文档open in new window；
  • 明白微信接口的组成，自有服务器、微信服务器、公众号（还有其它各种号）、测试号、以及通信原理（交互过程）；
  • 了解基本的 HTTP 协议，Header 头、请求方式（GET\POST\PUT\PATCH\DELETE）等；
  • 基本的 debug 技能，查看 PHP 日志，Nginx 日志等。
