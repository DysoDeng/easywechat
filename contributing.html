# 开始之前

请严格遵循以下代码标准:

  • PSR-2open in new window.
  • 使用 4 个空格作为缩进。

# 流程

  1. Fork overtrue/wechatopen in new window 到本地.
  2. 创建新的分支：
    $ git checkout -b new_feature
1
  1. 编写代码。
  2. Push 到你的分支:
    $ git push origin new_feature
1
  1. 创建 Pull Request 并描述你完成的功能或者做出的修改。

{warning} 注意：注释请使用英文
